📊 Edu Analytics Dashboard

JavaFX desktop application for managing students, courses, and predictions with charts and CSV storage.

✨ Features
Manage students, courses, and predictions
Add, delete, and import data
Bar & pie chart visualizations
Real-time UI updates
CSV-based data storage
⚙️ Requirements
Java 11+
JavaFX SDK configured
▶️ Run

If running outside Git Bash, add VM options:

--module-path "path-to-javafx/lib" --add-modules javafx.controls,javafx.fxml
💾 Storage

All data is stored locally in CSV files.
